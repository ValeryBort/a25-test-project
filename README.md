### Как запустить автотесты

#### Технические требования

 - Python 3.8
 - Pip

#### Настройка окружения

1. Установить зависимости 
    
        $ pip install -r requirements.txt

1. Установить playwright
   
        $ playwright install

1. Запустить тесты

        $ pytest --browser chromium --browser firefox
